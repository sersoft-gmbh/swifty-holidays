Calculator

public protocol Calculator

Represents a calculator that can calculate holidays using a fixed type of calendar.

  • Context

    The context that this calculator uses.

    Declaration

    Swift

    associatedtype Context : CalculationContext
  • calendar

    The calendar this calculator uses for its calculations.

    Note

    Calculators should always set the time zone of their calendar to UTC.

    Declaration

    Swift

    var calendar: Calendar { get }
  • context

    The current state of the context used by the calculator.

    Note

    This is just the current state in time. The context can change at any time and the value returned here might be outdated by the time it is returned.

    Declaration

    Swift

    var context: Context { get }
  • initialize(with:)

    (Re-)Initializes the context of the calculator with a new one, replacing the old one. This API is useful if you create a new calculator and want it to use the context of another one (or a deserialized context).
